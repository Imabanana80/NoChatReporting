# NoChatReporting
A simple plugin that disables chat reporting by replacing user sent messages with system messages

### How to use:
Just drag the NoChatReporting.jar into your server's /plugins folder and restart the server. The plugin will be automatically enabled and start working.

### Note:
This plugin is not compatable with most if not all, chat message alturing plugins, such as permissions plugins, rank plugins, and essentialsX
